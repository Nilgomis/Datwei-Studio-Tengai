# Controls

PLAYER 1
W: Move Up
S: Move Down
A: Move Left
D: Move Right
SPACE: Shoot
//SHIFT: Special

PLAYER 2
Up arrow key: Move Up
Down arrow key: Move Down
Left arrow key: Move Left
Right arrow key: Move Right
NUMPAD 0: Shoot
//NUMPAD 0: Special
